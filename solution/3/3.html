Solution: Retinal Variants

Written by Abigail Caron

We are given flavortext, several lines of capital letters, and a picture of some animal-like creatures. We first notice the use of "translation" and "sequence" in the flavortext, as well as the author's name being BLASToise. This is hinting that the presented strings are actually amino acid sequences, and the first step is to run them through the NCBI-hosted BLAST program online to see what they code for. If the flavortext is not read carefully, this step might be a difficult leap, but one could notice that substrings are often similar between sequences, and the non-amino acid letters are never used save for in position 1.

Reading the first letters down gives "INDEX NAMES BY LAST DI" which is unfortunately an incomplete clue phrase, and not enough to act on immediately. After BLASTing each sequence, we notice that all 18 of them code for the retinal protein Pikachurin. Looking more closely at the top hit (100% identity) for each BLAST, we see that each sequence comes from a different species of animal:

Noting each species down, we then look back at the 18 "players". Each of these pictures is of a specific Pokémon, but with small deviations from their standard form - for example, the Totodile is green, the Omanyte reddish, the Nidoran tan with small ears and a tail. Each Pikachurin sequence comes from a species of animal that matches one of the pokemon pictures. For example, Xatu in the image appears to have the heart-shaped face of a barn owl - and the 6th sequence from the top is also from a barn owl - Tyto alba. After matching each sequence with the corresponding Pokémon variant, reading the first letters of the sequences and then the first letter of the corresponding Pokémon in the same order gives "INDEX NAMES BY LAST DI/GIT TAXON IDS SORT LEN".

This is hinting that we should look up the taxon ID for each identified species, index into the name of the pokemon by the last digit of the taxon ID number, and then sort by the length of the amino acid sequences (each sequence has a unique number of residues from 140 to 157). To get the taxon ID, you can look on the NCBI Taxonomy Database and search by species name, or you can click on the BLAST result for each sequence and navigate to the taxon ID directly from there. As an example, for Xatu, the taxon ID of Tyto alba is 56313, so we would take the third letter of "Xatu" - the "T". When these letters are rearranged by length of the amino acid sequences 140-157, one can see the final answer: VICTORY CELEBRATION.
