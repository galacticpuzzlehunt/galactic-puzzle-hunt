When paired up by first letter, the length of the cryptic answer is always one more than the length of the normal answer. This, combined with the title, suggests that we can index the position of the space in the two-word normal answer into the one-word cryptic answer (for example, AT REST would pick the 3rd letter from ADDRESS). The resulting letters spell out DO IT AGAIN USING CRYPTIC CLUES.

The length of each cryptic clue, in words, is always one less than the length of its answer, and each one can be unambiguously divided into a definition half and a wordplay half. We can therefore index the position of the definition/wordplay split into the answer (for example, “Spot and then apply bandages: dissertation” consists of a 5-word wordplay half and a 1-word definition half, which means taking the 6th letter from ADDRESS). These letters spell out the final answer SECOND PLACE IS THE FIRST LOSER.
